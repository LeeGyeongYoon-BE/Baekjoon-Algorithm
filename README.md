# 🧮 Baekjoon Online Judge Solutions

> 백준(BOJ) 알고리즘 문제 풀이 저장소  
> Java | 단계별로 풀어보기 | 꾸준히 성장하는 알고리즘 기록

---

## 📚 About
이 저장소는 **백준 온라인 저지(BOJ)** 문제 풀이 코드를 정리한 공간입니다.  
매일 조금씩 문제를 풀며 **자료구조, 알고리즘, 구현력**을 향상시키는 것이 목표입니다.

---

## 🛠 Language & Environment
- **Language**: Java ☕  
- **IDE**: IntelliJ IDEA
- **Input Method**: `BufferedReader`, `StringTokenizer`  
- **Output Method**: `StringBuilder`
